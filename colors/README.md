# Ayu Colors for GitHub

Custom GitHub color schemes based on the Ayu theme, designed for use with the Stylus browser extension.

## Themes

| Theme | Description |
|-------|-------------|
| ayu-dark | Dark theme with warm orange accents |
| ayu-dark-green | Dark theme with green accents |
| ayu-light | Light theme with orange accents |
| ayu-mirage | Mirage (amber) theme |
| ayu-mirage-green | Mirage theme with green accents |

## Install with Stylus

[Stylus](https://addons.mozilla.org/firefox/addon/styl-us/) is a browser extension that lets you manage userstyles.

### Manual Install

1. Install the [Stylus extension](https://addons.mozilla.org/firefox/addon/styl-us/) for Firefox or [Stylus for Chrome](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfoboohticcbr)

2. Open the Stylus management window:
   - Firefox: Click the Stylus icon → "Manage installed styles"
   - Chrome: Click the Stylus icon → "Manage Styles"

3. Click "Write new style"

4. In the "Name" field, enter a name like "Ayu Dark"

5. In the "Applies to" section, choose "URLs starting with" and enter `https://github.com/`

6. Copy the content of the theme file you want (e.g., `ayu-dark`) and paste it into the editor

7. Click "Save" or press `Ctrl+S`

## Updating

To update a style, open the Stylus management window, click the edit icon next to the style, replace the content with the latest version, and click "Save".
